# Velvero SaaS MVP (Local Run)
1. Open PowerShell and go to project folder (after unzipping):
   cd velvero-saas-mvp

2. Install dependencies:
   npm install

3. Start the server:
   npm start

4. Open browser:
   http://localhost:3000

5. Upload sample_retail.csv or your own CSV to see KPIs, Insights, and charts.
